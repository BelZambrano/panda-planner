# 🐼 Panda Planner

Aplicación web desarrollada con Django para organizar metas de forma simple, visual y con un diseño "cute minimal".

---

## ✨ Descripción

Panda Planner es un proyecto enfocado en la gestión de metas personales, permitiendo crear, editar, eliminar y visualizar objetivos de manera clara y amigable.

El enfoque principal del proyecto fue combinar funcionalidad backend con una experiencia de usuario agradable y visualmente coherente.

---

## 🚀 Funcionalidades

- Crear nuevas metas  
- Visualizar lista de metas  
- Editar metas existentes  
- Eliminar metas  
- Interfaz con diseño "cute minimal"  
- Persistencia de datos con SQLite  

---

## 🛠 Tecnologías utilizadas

- Python  
- Django  
- HTML5  
- CSS3  
- SQLite

## ⚙️ Instalación y uso

1. Clonar el repositorio:

git clone https://github.com/TU-USUARIO/panda-planner.git  
cd panda-planner  

2. Crear entorno virtual:

python -m venv venv  

3. Activar entorno virtual (Windows):

venv\Scripts\activate  

4. Instalar dependencias:

pip install -r requirements.txt  

5. Ejecutar migraciones:

python manage.py migrate  

6. Ejecutar servidor:

python manage.py runserver  

7. Abrir en navegador:

http://127.0.0.1:8000/
